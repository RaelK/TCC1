# Atividades A1 e A2

## Atividade A1 - Correção Memorial Descritivo da Graduação
Esta atividade consiste na correção do memorial descritivo da graduação, com o objetivo de revisar e aprimorar o conteúdo, destacando as principais informações e aspectos da trajetória acadêmica.

## Atividade A2 - Fichamento: Área, Tema e Fundamentação
Nesta atividade, será realizado um fichamento abordando a área, o tema e a fundamentação, com o intuito de organizar as referências e criar uma base sólida para o desenvolvimento do trabalho.
